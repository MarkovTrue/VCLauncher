# <img src="https://github.com/user-attachments/assets/da615466-683e-4cc5-8380-32a98d743ba0" alt="Logo" width="32"/>&nbsp; VCLauncher

[![GitHub release](https://img.shields.io/github/release/MarkovTrue/VCLauncher)](https://github.com/MarkovTrue/VCLauncher/releases)

Удобный лаунчер для проекта [Video-compare](https://github.com/pixop/video-compare). 
Использует утилиту **ffprobe.exe** из пакета [FFmpeg](https://github.com/FFmpeg/FFmpeg).

* Позволяет сравнивать видео с разными пропорциями, без растягивания картинки, например 21:9 BDRip и 16:9 Open Matte
* Быстрый выбор горизонтального или вертикального сравнения
* Проверяет, чтобы окно не выходило за пределы экрана, если разрешение видео больше разрешения монитора
* Запоминает последние выбранные файлы и папки
